# sargasso
Which is the relationship between distance to coast line and DO in Chetumal Mexico?

The repository led to an amazing collaboration with [Nancy Cabanillas-Theran](https://www.ecosur.mx/academico/ncabanillas), Overlapping niches among sea urchin species reveal Sargassum-induced degradation at Caribbean shallow coral reefs published at: [https://doi.org/10.1016/j.scitotenv.2025.180297](https://doi.org/10.1016/j.scitotenv.2025.180297) and covered by the scidev news portal in Spanish [https://www.scidev.net/america-latina/news/invasion-del-sargazo-abre-oportunidades-para-la-ciencia/](https://www.scidev.net/america-latina/news/invasion-del-sargazo-abre-oportunidades-para-la-ciencia/).
